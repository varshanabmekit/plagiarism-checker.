# Plagiarism Checker (Python)

## 📌 Description
This project detects plagiarism between two text inputs using cosine similarity. It converts text into numerical vectors and compares them to calculate similarity percentage.

## 🛠️ Technologies Used
- Python
- Scikit-learn (CountVectorizer, Cosine Similarity)

## ⚙️ How It Works
1. Takes two text inputs from the user
2. Converts text into vectors using CountVectorizer
3. Computes similarity using cosine similarity
4. Displays similarity percentage

## ▶️ How to Run
1. Install required library:
   pip install scikit-learn

2. Run the program:
   python plagiarism_checker.py

## 📊 Example

Input:
Enter first document:
this project is about matrix

Enter second document:
this project explains matrix

Output:
Plagiarism Similarity: 75.0 %
